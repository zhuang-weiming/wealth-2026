### 💡 拍摄/制作建议 (Production Notes)

1.  **录屏工具**: 直接使用浏览器打开最终版的 HTML 文件，使用录屏软件（如 OBS 或 QuickTime）录制右侧手机模拟器的交互过程。
2.  **交互节奏**:
      * 在点击“Next Step”时，稍微停顿 1-2 秒，让观众看清 AI 的打字机效果和卡片弹出的动画。
      * 在 Face ID 弹窗出现时，配合音效（如“叮”的一声）。
3.  **视觉重点**:
      * 当 AI 发送 **[Insight Card]** (传导分析) 和 **[Product Card]** (产品卡片) 时，后期剪辑可以做放大处理（Zoom-in），强调这是原生 UI 组件，而不是简单的文字。
4.  **字幕**: 关键术语建议中英文对照，例如：*“传导机制 (Transmission Mechanism)”*、*“股债跷跷板 (Barbell Strategy)”*。
