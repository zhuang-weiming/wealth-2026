UC-04: Global Wealth Optimizer (全球财富优化器)

## Context

这是HSBC区别于任何本土银行或互联网券商的核心护城河。Premier客户的生活是全球化的（子女留学、海外置业、跨国工作），资金需要在多币种、多地区间流动。

## Problem Statement

用户在跨国资金管理上极度低效：看不准汇率时机，不懂各国复杂的转账额度限制，也不了解异地的高息理财机会。

## Goals

统一管理全球所有HSBC账户。

利用汇率波动和利差进行套利或对冲。

简化跨境资金链路。

## Non-Goals

协助洗钱或规避外汇管制。

提供极其冷门货币的投机建议。

## User Personas

Li (40岁，孩子在英国读书): 手持人民币和美元，需要定期支付英镑学费，对汇率敏感，希望降低换汇成本。

## User Journey

Monitor: 用户设定“9月支付£30k学费”。

Alert: 6月，AI监测到英镑兑美元跌至低点，且预测未来将反弹。

Advice: “Li，现在的汇率比您设定的预算低5%，建议立即购汇。”

Optimize: “购汇后，建议存入HSBC UK的3个月定存（年化5%），刚好在9月开学前到期，不仅锁定了汇率，还赚取了利息。”

### Screen Flow

Global Dashboard (世界地图显示各地资产分布)

FX Watchlist (汇率监控与目标价设定)

Smart Transfer Path (购汇 -> 转账 -> 理财 一站式路径)

Rate Arbitrage (全球利息地图：哪里的定存利率最高？)

## Key Features

全球统一视图: Global View数据底层打通。

智能汇率触发器: 结合技术面分析的汇率提醒。

跨时区/跨币种套利: 利用地区利差（Interest Rate Parity）推荐存款地。

## AI-Native Advantages

多维数据合成: 同时处理汇率趋势、各国利率政策、转账时效、手续费模型，瞬间计算出“最划算”的资金路径。
