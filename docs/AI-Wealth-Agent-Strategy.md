HSBC拥有庞大的"中产富裕"群体（卓越理财/尊尚），但这部分用户处于服务断层中：

资产规模：$100K-$2M（占客户总数60%，AUM占比35%）

服务现状：RM人均管理550人，实际深度服务<20人

竞争威胁：数字平台（Robinhood、Wealthfront）以低成本提供自动化服务

核心洞察
这个群体已完成从"赚钱"到"管钱"的转变：

财富积累阶段结束：精力仍在主业，资产管理零散滞后

专业服务需求：普通App无法处理复杂资产，私人银行成本过高

风险防范优先：从"增值"转向"安全"和"传承"

解决方案
利用Native-AI Agent，将私人银行思维框架与互联网银行数据洞察结合，提供：

全资产视角：股/债/房/现金的宏观配置

流动性管理：解决"资产富现金穷"问题

风险防范：税务优化、传承规划、全球配置

生活服务：权益管理、消费洞察、目标导航
